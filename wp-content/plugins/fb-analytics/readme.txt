== Installation ==

= using FTP or similar =
* unzip fb-analytics file and 
* Upload "fb-analytics" folder to the "/wp-content/plugins/" directory.
* Activate the plugin through the "Plugins" menu in WordPress.

= From Dashboard ( WordPress admin ) =
* plugins -> Add New
* search for 'wp shapes'
* click on Install Now  and then Active.

== Upgrade Notice ==

= using FTP or similar =
* Delete fb-analytics folder - your setting will not lost.
* unzip fb-analytics file and 
* Upload "fb-analytics" folder to the "/wp-content/plugins/" directory.
* Activate the plugin through the "Plugins" menu in WordPress.

= From Dashboard ( WordPress admin ) =
* If plugin new version released - you can see 'update now' link at wp-admin -> plugins
* click on 'update now'
